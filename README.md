# Hamlit::Block

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'hamlit-block'
```

And then execute:

    $ bundle

## Usage

Basically the same as Hamlit. Contents inside script block is not rendered but available via `yield`.
